# 造型製圖工具

將遊戲造型截圖切割、排列，並與封面明信片合併輸出成一張圖。

---

## 環境需求

- Python 3.8+
- pip

---

## 安裝

```bash
pip install flask pillow numpy
```

---

## 啟動

```bash
python app.py
```

開啟瀏覽器前往 `http://localhost:5000`

---

## 使用流程

### 1. 封面
上傳明信片封面圖片，最終輸出時會放在最上方。

### 2. 造型圖片切割

上傳一張或多張造型截圖，並設定切割參數：

| 欄位 | 說明 |
|------|------|
| 起始 X | 跳過左側導覽列的像素寬度，紅線左側會被忽略 |
| 行 N → Y 起點 | 該行在圖片中的 Y 座標起點（藍線位置） |
| 行 N → 欄數 | 該行有幾張造型 |
| 行 N → 起始 X | 可覆蓋每行各自的左側起點 |

**支援的截圖格式：**
- 單排截圖（例如造型列表頁，一排 5 或 6 張）→ 只設一行
- 多排截圖（一張圖有上下兩排）→ 按「＋ 新增一行」，手動設定分界 Y 座標
- 多張截圖批次上傳 → 每張都套用相同的行設定

設定完成後按「**✂ 切割並載入**」，切好的造型會出現在左側清單。

> **注意：** 截圖請使用遊戲內「造型列表」頁面，卡片要完整可見。  
> 登入畫面拼接截圖、UI 元素遮住卡片的圖片無法正確切割。

### 3. 格子排版

1. 設定**輸出列數**與**輸出欄數**，按「建立格子」
2. 按「**自動填入**」將左側造型依序填入格子
3. 手動調整：
   - **拖曳**左側清單的造型到任意格子
   - **格子之間拖曳**互換位置
   - **點選格子**（紅框）後按 `Backspace` 清空該格

### 4. 輸出

按「**產生圖片**」，預覽圖會顯示在下方，確認後按「**下載**」儲存。

輸出格式：JPEG，寬度 1200px，封面在上、格子表格在下。

## 常見問題

**Q: 切出來的造型位置跑掉？**  
調整「起始 X」，把紅線對齊造型卡片的左邊緣，再按「重新切割」。

**Q: 多排截圖只切到一排？**  
按「＋ 新增一行」，把藍線拖到兩排的分界處，每行各自設定欄數。

**Q: 上傳多張截圖會怎樣？**  
每張都套用相同的行/欄設定切割，全部的造型會累積到左側清單。

**Q: uploads 資料夾越來越大？**  
手動清空 `uploads/` 和 `outputs/` 即可，不影響程式運作。
